This respository is intended to accompany the data paper Kennedy et al., 2023 and the MOCHA synthesis dataset, available here: https://doi.org/10.25921/2vve-fh39. Here, you can find code to reproduce all figures in the paper and supplement, examples of data manipulation, and examples of data formatting and processing.
